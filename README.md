# java-problem-solving3
Problem Statement

Given two strings haystack and needle, return the index of the first occurrence of needle in haystack. If needle is not part of haystack, return -1.
Example 1

Input:

haystack = "sadbutsad"
needle = "sad"

Output:

0

Explanation:
The substring "sad" appears at indices 0 and 6. The first occurrence is at index 0.

Example 2

Input:

haystack = "leetcode"
needle = "leeto"

Output:

-1

Explanation:
The substring "leeto" is not present in "leetcode".
